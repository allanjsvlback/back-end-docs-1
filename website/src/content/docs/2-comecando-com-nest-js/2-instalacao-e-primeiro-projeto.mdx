A maneira mais rápida e recomendada de começar um projeto Nest.js é utilizando o **Nest.js CLI** (Command Line Interface). O CLI é uma ferramenta poderosa que não só cria a estrutura inicial do projeto, mas também ajuda a gerar os diferentes blocos de construção da sua aplicação (módulos, controllers, etc.), garantindo a padronização e a produtividade.

## Pré-requisitos

Antes de começar, certifique-se de que você tem o Node.js instalado na sua máquina.

-   **Node.js:** Versão 16 ou superior.

Se você não tiver o Node.js, pode baixá-lo em [nodejs.org](https://nodejs.org/). A instalação do Node.js também inclui o gerenciador de pacotes **npm**.

## Passo 1: Instalando o Nest.js CLI

Para instalar o CLI globalmente na sua máquina, abra seu terminal e execute o seguinte comando:

```bash
npm i -g @nestjs/cli
```

A flag `-g` garante que o comando `nest` se torne disponível em qualquer diretório do seu sistema.

<Aside title="Prefere não instalar globalmente?">
Se você não quiser instalar pacotes globalmente, pode usar o `npx`, que vem com o npm, para executar o CLI diretamente:

```bash
npx @nestjs/cli new meu-projeto
```
Esta abordagem baixa e executa o comando sem instalá-lo permanentemente no seu sistema.
</Aside>

## Passo 2: Criando um Novo Projeto

Com o CLI instalado, criar um novo projeto é muito simples. Navegue até o diretório onde você deseja criar seu projeto e execute:

```bash
nest new meu-primeiro-projeto
```

O CLI fará algumas coisas por você:
1.  Ele perguntará qual gerenciador de pacotes você prefere usar (`npm`, `yarn` ou `pnpm`).
2.  Criará um novo diretório chamado `meu-primeiro-projeto`.
3.  Gerará a estrutura de arquivos e pastas com um código inicial (boilerplate).
4.  Instalará todas as dependências necessárias.
